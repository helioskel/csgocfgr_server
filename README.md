# csgocfgr-server

***NOTE: This project is no longer being maintained. This is a simple rails 5 JSON API server for persisting data from the csgocfgr-client application.***

## Ruby version

* `ruby 2.3.1`

## System dependencies

* `postgresql >= 9.6.1`

## Database creation

* `rails db:create`
* `rails db:migrate`

## Running

* `rails server`
* Navigate to `http://localhost:3000/api/v1/cfgs`
